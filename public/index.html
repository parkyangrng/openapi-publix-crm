API Endpoints

13 POST + 2 GET

Every API endpoint accepts POST with Content-Type: application/json

POST
/api/customers
List customers — filter by tier, store, status
{ tier, store_id, active, page, limit }
POST
/api/customers/profile
Full customer profile with tier details and spend total
{ customer_id* }
POST
/api/customers/search
Search customers by name or email
{ q*, tier, page, limit }
POST
/api/points/balance
Points balance, tier multiplier, perks, next tier progress
{ customer_id* }
POST
/api/points/earn
Award points for a purchase — applies tier multiplier
{ customer_id*, purchase_amount*, order_id }
POST
/api/points/redeem
Redeem points for dollar credit (min 100 pts = $1.00)
{ customer_id*, points* }
POST
/api/points/tiers
All loyalty tier definitions with perks and multipliers
{ }
POST
/api/orders
List orders — filter by customer, store, status
{ customer_id, store_id, status, page, limit }
POST
/api/orders/detail
Full order with line items, tax, points earned/redeemed
{ order_id* }
POST
/api/history/summary
Spending summary — total orders, avg order, favorite category
{ customer_id* }
POST
/api/history/by-category
Spend breakdown ranked by category
{ customer_id* }
POST
/api/history/top-products
Most purchased products by qty with total spend
{ customer_id*, limit }
POST
/api/products
Product catalog with category and points-per-dollar rate
{ category, min_points_ppd }
GET
/api/openapi.json
OpenAPI 3.1 machine-readable spec

Example requests

curl examples -X POST
Customer profile
curl -X POST https://publix-crm-api.onrender.com/api/customers/profile \
  -H "Content-Type: application/json" \
  -d '{"customer_id":"CUS-001"}'
Points balance + tier info
curl -X POST https://publix-crm-api.onrender.com/api/points/balance \
  -H "Content-Type: application/json" \
  -d '{"customer_id":"CUS-002"}'
Earn points for purchase
curl -X POST https://publix-crm-api.onrender.com/api/points/earn \
  -H "Content-Type: application/json" \
  -d '{"customer_id":"CUS-001","purchase_amount":45.99}'
Order detail with line items
curl -X POST https://publix-crm-api.onrender.com/api/orders/detail \
  -H "Content-Type: application/json" \
  -d '{"order_id":"ORD-2026-0003"}'
Purchasing summary
curl -X POST https://publix-crm-api.onrender.com/api/history/summary \
  -H "Content-Type: application/json" \
  -d '{"customer_id":"CUS-002"}'
Spend by category
curl -X POST https://publix-crm-api.onrender.com/api/history/by-category \
  -H "Content-Type: application/json" \
  -d '{"customer_id":"CUS-001"}'
Top purchased products
curl -X POST https://publix-crm-api.onrender.com/api/history/top-products \
  -H "Content-Type: application/json" \
  -d '{"customer_id":"CUS-002","limit":5}'
